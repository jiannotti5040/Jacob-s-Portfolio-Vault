# Individual Programs

**Author & sole owner: Jacob Iannotti. Proprietary — all rights reserved.**

Standalone programs and theory that stand on their own, outside the main
certification line.

### Programs

- **Video Transcription Program** (`transcribe_final.py`) — a working,
  fully-automated pipeline written for a family member's business: it converts
  ~100 hour-long meetings into single, speaker-attributed minutes. Built around
  the operator as the person in the middle balancing compute and size limits.

  ```bash
  python3 transcribe_final.py            # run the pipeline (see file header for inputs)
  ```

### Theory

- **PIH — Unified Theory** (`PIH_UNIFIED_THEORY.md`) and **PIH — Closed Theory**
  (`PIH_CLOSED_THEORY.md`) — variational principles for dynamical systems with
  *endogenous* uncertainty (uncertainty generated by the system itself rather
  than imposed from outside).

### Early experiments

- **Throughput Semantic Engine** (`Thruput_Compiled.txt`) — an early experiment
  in standardizing information mediums (video, audio, music, language) into one
  semantic space. It was doomed to fail in its original form, but it was a strong
  semantic start and the codebase remains useful scaffolding.

Each item here is independent of Chiron and the certification program; they are
collected as the body of individual work.
