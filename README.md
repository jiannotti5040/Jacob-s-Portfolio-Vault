The overall goal was to see if you can let intellectual and logical fleet fly the right mathematical artifact might already be realized and just in need of bespoke vocational placement. I had to Departitionlize mathematical convention and order and source from disparate mathmatics- this is the outcome. It's accuracy I hope but make no claims to.
