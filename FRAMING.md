# Framing — the same engines, two vocabularies

Framing is a dial, not a switch. Every capability below is the *same code, the same
tests, the same numbers*; only the title changes with the audience. The public
default is the civilian column; the contractor column is for targeted defense /
national-security conversations.

| Capability (file) | Civilian framing | Contractor / defense framing |
|---|---|---|
| Exact recovery (`chiron.py`) | Exact structure recovery with honest abstention | Signal/pattern extraction & classical cryptanalysis, zero false-positive |
| Ambiguity measurement (`infectatrum_bridge.py`) | Spec/contract ambiguity & coherence measurement | Engineered-multiplicity / adversarial-input detection |
| Origin signature (`--signatures`) | Provenance & source attribution | Source attribution & deception detection |
| Adversarial detector (`adversarial_score`) | Synthetic-inflation / anomaly detection | Channel-flood / deliberate-noise discrimination |
| Robustness coupling (`uma_bridge.py`) | Robustness of recovered structure under perturbation | Signal-integrity / structural stress test under noise & jamming |
| Certify a finding (`certify_finding.py`) | High-stakes decision assurance | Daubert-admissible, attested autonomous-decision certificate |
| Governance gate (`govern.py`) | AI standard-of-care / duty-of-care gate | Rules-of-engagement / policy-isolation gate with escalation |
| Structured intake (`intake.py`) | Multi-source data standardizer | Multi-feed standardizer to a common decision schema |
| Value/coherence (`ontological.py`) | Value & coherence scoring of evidence | Mission-value & concordance scoring across findings |

## How to deploy the dial

- **Public README / portfolio:** civilian vocabulary (the default everywhere).
- **Targeted outreach:** this file (or a private-grow variant) carries the contractor
  vocabulary; nothing in the code changes — you re-title sections for the audience.

The mechanism is genuinely dual-use, and that is an asset: a Daubert-admissible,
ROE-aware certifier is *more* compelling to a defense contractor framed as such, and
to a hospital or bank framed as "high-stakes decision assurance." The engineering is
one thing; the framing meets the reader where they are.
