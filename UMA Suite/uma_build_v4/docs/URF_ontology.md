# URF as Ontology

**The Universal Resonance Framework as a domain-independent grammar.**

This document treats URF/UMA/RSLS not as a physics theory, but as a
*controlled vocabulary* with a typed grammar that can be instantiated
in multiple domains. The physics instantiation (Stages 1–6) is one
realization. Economics is another. AI safety is a third. The shared
structure is the ontology.

The ontology is not a metaphor. The same mathematical objects, with
the same algebraic relations, recur across domains. The claim is
structural: harmony-seeking dissipation with skew-symmetric regime
generation is what these systems *are*, not what they *resemble*.

## 2. Axioms

The ontology rests on five axioms. Each is domain-independent.

**A1. Convex coercivity.** 𝓕 is convex and coercive: its Hessian is positive-definite and ‖**u**‖ → ∞ ⇒ 𝓕[**u**] → ∞. Equivalently: H is concave with a unique maximum.

**A2. Singular barrier.** There exists a critical surface ∂𝓜 ⊂ 𝓜 such that V′(d(**u**, ∂𝓜)) → ∞ as **u** → ∂𝓜. The barrier enforces the domain: the dynamics cannot cross ∂𝓜.

**A3. Skew–gradient decomposition.** The dynamics admit a GENERIC decomposition d**u**/dt = ∇H + Δ·**J**(**u**), where ∇H ⊥ **J** in the metric induced by ∇²𝓕.

**A4. Causal transport.** Information about the state propagates with finite speed: c_diff = √(μ/(τ_J τ_M)) ≤ c_geom, the Whitham subcharacteristic condition.

**A5. Anosov closure.** If β^φ ≠ 0 with non-trivial gradient, the cone aperture ΔΛ > 2c_diff strictly, and the flow admits a uniformly hyperbolic invariant set on which the SRB measure is well-defined. This is the regime-transition mechanism.

## 3. Derived relations

From A1–A5 a number of structural theorems follow that hold in any instantiation:

**T1. Bridge.** H_URF = −𝓕. Harmony-maximization and free-energy-minimization are the same gradient flow viewed from opposite sides (April 8 2026 derivation).

**T2. Sign-flip duality.** Division-algebra signature (harmony) and split-algebra signature (disharmony) are related by metric signature flip. "Disharmony isn't the absence of harmony — it's harmony under hyperbolic shear." Tonally: stable rotation vs. dissipative shear, both structurally present.

**T3. Regime trichotomy.** The sign of Δ partitions the dynamics:
- Δ > 0: singular regime (saturation-bound, blowup-bound)
- Δ < 0: oscillatory regime (limit-cycle behavior)
- Δ = 0: diffusive regime (pure gradient flow, monotone relaxation)

**T4. Emergent scale.** ℓ_* = √(μ τ_M / V″(M)) is mesh-independent: it survives the dx → 0 limit and is therefore an intrinsic property of the system, not a numerical artifact.

**T5. Quantum statistics from chaos.** The Born rule (P_i = ∫|Ψ|²dΓ) is the ergodic limit of the SRB measure on the chaotic attractor generated by Δ ≠ 0. This is not a postulate but a derivation given A5.

**T6. Detectability bound.** The macroscopic dissipation length scale ℓ_* must satisfy log(M/ℓ_*) ≤ −log(N_floor) / (2 M Γ) to remain observationally distinguishable from a no-substrate null. This is the "Macroscopic Mandate."
