How to run Chiron

  1. Put chiron.py in any folder (it can sit beside the data you want it to read).
  2. Start this console: python3 chiron.py serve — open http://127.0.0.1:8765. No installs needed.
  3. To save memory as a JSON file in that folder: python3 chiron.py serve --congress chiron_memory.json.
  4. Headless: python3 chiron.py run reads the whole current folder; run /path reads any folder.

How Chiron grows — the exact points

A new DOMAIN is bornthe first time a structurally-distinct kind of material is seen (numbers vs text vs code vs graph). The same kind extends it.
A new ORGANISM is mintedwhen a reasoning cycle meets structure its charts can't cleanly read and the proposed capability passes the gate: non-colliding, runnable, still able to reject.
Integral vs generala verified, reusable generator is integral and joins the law library; everything else is retained as knowledge. The gate is the discernment.

Self-growth — on a leash

Chiron can grow itself in three steps, each one safe to undo. It can get smarter, but by construction it cannot take anything away.

1 · New concepts, from proof onlyWhen the same proven rule shows up in two or more subjects, Chiron names it a concept — an idea it earned. It never invents concepts it hasn't proven, so it can't drift.
2 · Proposed code changes, quarantinedIt can write a change to its own source, but the change is parked in a proposals/ folder and is never applied on its own.
3 · Applied only if it stays soundA change is applied only after a reversible backup is taken and the changed version still passes the full self-test. Otherwise it's refused. Unattended self-edit must be explicitly switched on (CHIRON_ALLOW_SELF_EDIT=1); off by default.

Glossary

Plain definitions for the words used across this console.

collapseTake a surface (anything codified) and recover the smallest rule that reproduces it. The core move.
surfaceAny input with structure: a number sequence, a cipher, code, a grid of letters, text with figures in it.
invariant / generatorThe rule underneath a surface — the thing that stays true and can regenerate it. The generator is the exact recovered rule.
verifiedThe rule was checked by predicting parts of the input it was never shown, and got them exactly right. A proof, not a guess.
integral vs generalIntegral: a verified, reusable rule — it joins the law library. General: kept as knowledge, but no proven rule was found.
domainA subject Chiron sorts material into (mathematics, law, biology…), so its memory is organized rather than one big pile.
CongressChiron's whole memory: its domains, its law library, and its provenance record, bound by one signed root.
articulateThe reverse of collapse: take a recovered rule and speak it back — regenerate or extend the original, or re-voice it in a new vocabulary.
cross-domain transferThe same rule appearing in a second subject — evidence the knowledge generalizes.
conceptA rule proven in two or more subjects, named and kept as an idea Chiron earned.
membraneA gate on a domain. A sealed domain won't let its rules leak outward — selectivity by design.
certificateThe plain report Chiron issues for a finding: what it found, why it believes it, and what would prove it wrong.
